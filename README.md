# TDental Bot Assets

Repositorio para almacenar los recursos públicos utilizados por el chatbot.

## Estructura

- `/pdfs` → Archivos PDF utilizados por el chatbot.
- `/flyers` → Imágenes, flyers y otros recursos gráficos.

Los archivos publicados estarán disponibles mediante:

https://pdfs.tdental.mx/

La estructura de carpetas podrá ampliarse conforme crezcan las necesidades del proyecto.
